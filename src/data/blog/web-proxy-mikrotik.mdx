## Pendahuluan

MikroTik RouterOS adalah sistem operasi jaringan yang populer dan sering digunakan untuk membangun jaringan kecil hingga besar. Salah satu fitur yang disediakan oleh RouterOS adalah Web Proxy, yang dapat digunakan untuk memblokir akses ke situs web tertentu dan memblokir file ekstensi yang tidak diinginkan, serta melakukan Transparent Proxy. Dalam artikel ini, kami akan membahas cara mengkonfigurasi Web Proxy di MikroTik RouterOS untuk memblokir situs web dan file ekstensi, serta melakukan Transparent Proxy.

![Thumbnail](https://ik.imagekit.io/taufik/blog/web-proxy-mikrotik/thumbnail.png 'Thumbnail')

Sebelum melakukan konfigurasi ini, pastikan kalian memiliki perangkat Mikrotik atau kalian bisa menggunakan aplikasi simulasi seperti PnetLAB atau GNS3, kalian juga bisa menginstall Mikrotik RouterOS pada virtual machine dengan VirtualBox Oracle.

Pertama-tama, kalian perlu melakukan konfigurasi dasar sampai LAN mendapatkan koneksi internet. Jika kalian belum melakukan konfigurasi dasar Mikrotik kalian bisa mengikuti cara konfigurasinya di tutorial ini [Cara Konfigurasi Dasar Mikrotik Menggunakan Winbox Untuk Pemula](https://taufikcrisnawan.dev/blog/konfigurasi-dasar-mikrotik).

## Konfigurasi Web Proxy

### Aktifkan Web Proxy pada MikroTik RouterOS

Langkah pertama dalam konfigurasi Web Proxy pada MikroTik adalah mengaktifkan fitur tersebut pada router. Untuk melakukannya, buka menu <Kbd>IP</Kbd> pada menu utama MikroTik dan pilih <Kbd>Web Proxy</Kbd>. Kemudian, centang kotak <Kbd>Enabled</Kbd> untuk mengaktifkan Web Proxy.

Kalian juga bisa mengisi Cache Administrator untuk menunjukkan Administrator jika client membutuhkan akses spesial.

Tekan <Kbd>Apply</Kbd> untuk melakukan perubahan pada konfigurasi Web Proxy.

![Enable Web Proxy](https://ik.imagekit.io/taufik/blog/web-proxy-mikrotik/enable-web-proxy.png 'Enable Web Proxy')

### Blokir Situs Web dengan Web Proxy

Setelah mengaktifkan Web Proxy, langkah selanjutnya adalah memblokir situs web tertentu. Untuk melakukan hal ini, buka menu <Kbd>Web Proxy</Kbd> pada MikroTik dan pilih tab <Kbd>Access</Kbd>. Kemudian, klik tombol <Kbd>Add</Kbd> untuk menambahkan aturan baru.

Dalam aturan ini, kalian dapat memilih untuk memblokir situs web berdasarkan nama domain, URL, atau alamat IP. Misalnya, jika kalian ingin memblokir situs web dengan nama domain **"example.com"**, kalian dapat memasukkan nama domain tersebut ke dalam kolom **"Domain Name"**. kalian juga dapat memilih untuk memblokir situs web berdasarkan kata kunci tertentu pada URL dengan memasukkannya ke dalam kolom **"URL"**.

Selain itu, kalian dapat memilih untuk memblokir situs web dengan mengaktifkan opsi **"Block File Extension"**. Dalam opsi ini, kalian dapat memilih jenis file ekstensi yang ingin kalian blokir, seperti **"\*.exe"** atau **"\*.mp3"**. Setelah memilih opsi yang sesuai, klik tombol <Kbd>Apply</Kbd> untuk menyimpan aturan tersebut.

### Konfigurasi Transparent Proxy

Selain memblokir situs web secara manual, kalian juga dapat menggunakan Transparent Proxy untuk memblokir situs web secara otomatis. Transparent Proxy memungkinkan kalian untuk memaksa semua koneksi HTTP yang masuk melalui router untuk melewati Web Proxy tanpa konfigurasi klien yang rumit.

Untuk mengaktifkan Transparent Proxy pada MikroTik, buka menu <Kbd>IP</Kbd> dan pilih <Kbd>Firewall</Kbd>. Kemudian, klik tab <Kbd>NAT</Kbd> dan tambahkan aturan baru dengan mengklik tombol <Kbd>Add</Kbd>.

Dalam aturan baru ini, kalian harus mengatur koneksi yang masuk ke port HTTP untuk melewati Web Proxy. Untuk melakukannya, masukkan informasi berikut:

- Chain: dstnat
- Protocol: tcp
- Dst. Port: 80
- Action: redirect
- To Ports: 8080
- Web Proxy: yes

Setelah aturan telah disimpan, semua koneksi HTTP yang masuk melalui router akan melewati Web Proxy secara otomatis.

### Tes Konfigurasi Web Proxy

Setelah menyelesaikan konfigurasi Web Proxy dan Transparent Proxy, tes apakah semua aturan yang telah dibuat bekerja dengan baik. Untuk melakukannya, buka browser web dan coba mengakses website yang telah diblokir tadi, jika berhasil akan muncul tampilan seperti dibawah:

![Tes Konfigurasi](https://ik.imagekit.io/taufik/blog/web-proxy-mikrotik/tes.png 'Tes Konfigurasi')

## Kesimpulan

Kesimpulan dari artikel ini adalah bahwa konfigurasi web proxy di MikroTik RouterOS dapat membantu kalian untuk memblokir akses ke website tertentu dan memblokir file ekstensi tertentu. Selain itu, fitur transparent proxy juga dapat digunakan untuk meningkatkan efisiensi dan kecepatan akses internet. Namun, kalian perlu melakukan konfigurasi dengan benar agar tidak terjadi masalah seperti memblokir akses ke situs yang sebenarnya diperlukan atau mengalami masalah jaringan yang tidak diinginkan. Dengan melakukan konfigurasi web proxy yang tepat, kalian dapat memanfaatkan fitur ini untuk meningkatkan keamanan dan efisiensi jaringan mereka.
